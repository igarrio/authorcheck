## AuthorCheck
```
AuthorCheck - телеграм-бот, розроблений для адміністраторів український каналів в месенджері Telegram.
Бот виконує перевірку певного художника за власною базою даних та вказує, чи доцільно займатися поширенням його робіт.

База розроблялась на добровільній основі небайдужими українцями
```
